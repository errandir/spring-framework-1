This part of the documentation covers support for Servlet-stack web applications built on the
Servlet API and deployed to Servlet containers. Individual chapters include xref:mvc:mvc.adoc#mvc[Spring MVC],
xref:mvc:mvc-view.adoc#mvc-view[View Technologies], xref:mvc:mvc-cors.adoc#mvc-cors[CORS Support], and xref:websocket:websocket.adoc#websocket[WebSocket Support].
